# guide de la maison d'ars-en-ré

Site statique prêt pour GitHub Pages.

## mise en ligne

1. Ouvrir le dépôt GitHub.
2. Cliquer sur Add file > Upload files.
3. Glisser tout le contenu de ce dossier à la racine du dépôt.
4. Cliquer sur Commit changes.
5. Aller dans Settings > Pages et publier depuis la branche main, dossier /(root).

## modification

- La page d'accueil est dans `index.html`.
- La fiche d'arrivée est dans `arriver.html`.
- Le style est dans `css/style.css`.
- Les images sont dans `images/`.
